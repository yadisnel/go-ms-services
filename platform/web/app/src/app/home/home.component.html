The simplest way to build, share and collaborate on microservices

Micro empowers developers to build microservices in the cloud and beyond without thinking about the underlying infrastructure.


Micro is built with a developer first focus.



What we're building

M3O

(pronounced "em-3-oh")

A global microservices platform
M3O is a serverless platform for microservices development in the cloud and beyond. For the past few years cloud-native technology has added significant complexity to software development and largely increased the barrier to entry for developers to get started.

“Building cloud-native services should be
easier not harder”

Small teams and individual developers must be empowered to build software at an accelerated pace but instead are forced to reason about infrastructure and operations.


Micro is redefining the way we build software by focusing entirely on the needs of developers and providing an environment in which we can realise the true potential of the cloud.
